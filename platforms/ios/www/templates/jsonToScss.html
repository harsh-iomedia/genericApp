To convert the JSON response into the Sass theming file follow the steps.

  1. There is a gulp task in gulpfile.js, named under 'jsonToSass'.
  2. Run the CMD gulp jsonToSass
  3. The Sass complied file will be created in 'scss' folder. After that you need to import the file in your main '.scss' file using the @import of Sass. Like. @import "theme";. The theme is the name of the '.scss' generated by gulp task that you run in step 1.


Note: You can change the URL from where you want to fetch the JSON in gulpfile.js.
