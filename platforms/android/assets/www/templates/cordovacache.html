Cordova Cache

This is a WebView cache plugin for Phonegap 3.3.0 / Cordova 3.3.1 supporting Android (>=2.3.3) and iOS(>=6.0). It allows to clear the cordova webview cache.

Purpose

We modified the plugin to delete cache db which is present in android and can only be access on rooted phones.

Installation

cordova plugin add https://github.com/harsh-iomedia/cordova-cache.git
