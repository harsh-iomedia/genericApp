Danwilson Google Analytics

Cordova (PhoneGap) 3.0+ Plugin to connect to Google's native Universal Analytics SDK

Installation

To add this plugin just type:
cordova plugin add https://github.com/danwilson/google-analytics-plugin.git

or Cordova 5 + cordova plugin add cordova-plugin-google-analytics

or for < Cordova 5.0 cordova plugin add com.danielcwilson.plugins.googleanalytics

For futher details follow the Github link
