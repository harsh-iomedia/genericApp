Phonegap Pushplugin

This plugin is for use with Cordova, and allows your application to receive push notifications on Amazon Fire OS, Android, iOS, Windows Phone and Windows8 devices.

Purpose

We modified the plugin to support the notification icon on Android 5+. Doing so you can use this plugin 2 ways.

  1. Create a png icon file with named ic_notify.png and place it in following places
    • resources/android/icon/ic_notify.png
    • platform/android/res/drawable/ic_notify.png
    • platform/android/res/drawable-hdpi/ic_notify.png
    • platform/android/res/drawable-ldpi/ic_notify.png
    • platform/android/res/drawable-mdpi/ic_notify.png
    • platform/android/res/drawable-xhdpi/ic_notify.png
    • platform/android/res/drawable-xxhdpi/ic_notify.png
    • platform/android/res/drawable-xxxhdpi/ic_notify.png
  2. Otherwise use the default icon of the application by changing in the plugin code-base at location(file path): plugins/com.phonegap.plugins.PushPlugin/src/android/com/plugin/gcm/GCMIntentService.java at line no:130 drawable/ic_notify to drawable/icon

Installation

cordova plugin add https://github.com/harsh-iomedia/phonegap-pushplugin.git
